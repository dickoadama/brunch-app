# Images Thématiques pour BRUNCH

## Instructions pour ajouter des images

1. Téléchargez des images gratuites sur des sites comme :
   - [Unsplash](https://unsplash.com)
   - [Pexels](https://pexels.com)
   - [Pixabay](https://pixabay.com)

2. Images recommandées :
   - `bg-hero.jpg` : Image de fond principale (intérieur de restaurant ou plat de brunch)
   - `recettes-hero.jpg` : Image pour la section recettes
   - `menu-hero.jpg` : Image pour la section menu
   - `clients-hero.jpg` : Image pour la section clients
   - `equipe-hero.jpg` : Image pour la section employés

3. Formats recommandés :
   - JPG ou PNG
   - Résolution minimum : 1200x800px
   - Taille de fichier optimisée pour le web (< 500KB)

4. Placez les images dans ce dossier (`public/images/theme/`)

## Exemples de recherches recommandées

- "brunch restaurant interior"
- "breakfast food photography"
- "cafe interior design"
- "professional kitchen"
- "restaurant staff team"

Les images seront automatiquement utilisées dans l'application une fois placées dans ce dossier.
